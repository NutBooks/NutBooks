# Bookmarks

## Frontend

Vue.js + Vite + TailwindCSS

### Prerequisites

- node v18
    - You can use nvm for convenience
        - <https://github.com/nvm-sh/nvm>

```shell
nvm use # node v18
cd frontend
npm install
npm run format
npm run lint
npm run dev
```

## API

### Prerequisites

cd api
